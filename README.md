Bienvenido al repositorio de mi portafolio personal. Este proyecto es una vitrina de mis habilidades como **Diseñador y Desarrollador Frontend**, construido con tecnologías web puras para garantizar ligereza, velocidad y un control total sobre el diseño.

🚀 **Live Demo:** [vsdesigner.com.ar](https://vsdesigner.com.ar/)

---

## ✨ Características Principales

- **Vanilla Stack:** Desarrollado 100% con HTML, CSS y JS, sin librerías externas pesadas.
- **Diseño Mobile-First:** Totalmente responsivo, adaptándose a cualquier tamaño de pantalla.
- **Interacciones con JS:** Manipulación del DOM para efectos de navegación, scroll suave y animaciones de entrada.
- **Estética Minimalista:** Uso estratégico de espacios, tipografía y modo oscuro para resaltar los proyectos de diseño.
- **Optimización SEO:** Estructura semántica para una mejor indexación en buscadores.

## 🛠️ Detalle Técnico

- **HTML5:** Estructura semántica y accesible.
- **CSS3:** Flexbox y Grid Layout para el diseño, junto con variables CSS para una gestión de colores eficiente.
- **JavaScript (ES6+):** Lógica para el menú interactivo, validaciones y animaciones de scroll (ScrollReveal o similares).
